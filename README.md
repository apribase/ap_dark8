# ap_dark8

ap_dark8 は 8色 (16色) しか色を扱うことができない Mac Visor (Terminal.app) 向けに作成した色設定です。
通常のターミナル操作および vim によるプログラミングを8色でもそれなりにできるように作成しました。

## Terminal.app の設定

<a href="http://github.com/apribase/ap_dark8/blob/master/Visor_AP_Dark8.terminal">Visor_AP_Dark8.terminal</a> は Mac の Terminal.app の設定をエクスポートしたファイルです。
ダブルクリックすることでターミナル設定に Visor_AP_Dark8 を追加することができます。
Visor のデフォルト設定に対して色を変更しているだけです。
Visor は起動時に Visor 設定を読み込むため、起動後すぐに読み込ませる場合は名前を Visor に変更してください。
